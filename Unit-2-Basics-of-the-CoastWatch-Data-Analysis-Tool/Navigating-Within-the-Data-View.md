# Navigating Within the Data View

You can navigate within the data view using the navigation buttons built into the data view. Use  <img src="../images/selection_view_onscreen.png" alt="" height="24" align="top">  **Zoom to selection** (click and drag),  <img src="../images/zoom_in_onscreen.png" alt="" height="24" align="top">  **Zoom in**, or  <img src="../images/view_1_1_onscreen.png" alt="" height="24" align="top"> **Zoom to actual size** to magnify the data view. Try zooming in on the Eastern Mediterranean Sea:

![CDAT window showing sea surface temperature data in the Eastern Mediterranean Sea](../images/cdat_zoom_in.png)

Try panning to the northwest by dragging on the data view:![CDAT window showing sea surface temperature data near Greece and Turkey](../images/cdat_pan.png)

Use  <img src="../images/zoom_out_onscreen.png" alt="" height="24" align="top">  **Zoom out**,  <img src="../images/fit_to_size_onscreen.png" alt="" height="24" align="top">  **Fill window**, or  <img src="../images/home.png" alt="" height="24" align="top">  **Fit image to window** to zoom back out. Try filling the data view window:

![CDAT window showing sea surface temperature data in the Red Sea](../images/cdat_fit.png)

###  <img src="../images/address_book.png" alt="" height="24" align="top">  Bonus exercises:

  - Try using **Full Screen Mode** accessed from the **View** menu, and use the controls to zoom and pan around the data view. Click the **X** at the top left to exit full screen mode.
  - Try changing the data view size either by dragging on the CDAT window edges, or you can use the **Window size** options in the **View** menu. This helps keep the view size the same to more easily export and compare images.
  - Watch a [YouTube video](https://youtu.be/CHUw7wCLBtU?list=PL_-bsOLKMYJybI8chOl90HWWd_jTsaO3e) on basic CDAT components and navigating around in the data view (the YouTube video covers an older version with slightly different view navigation controls).
