# Unit 2 Quiz Answers

### Question 1

In the CoastWatch Data Analysis Tool (CDAT), you can: (choose all answers that apply)

- [ ] Only load one variable at a time

- [x] Load multiple variables at a time

- [x] View data files in multiple tabs

- [x] Drag and drop a data file into the window to open

**General feedback:** Data files can be opened by dragging them into the CDAT window, and selecting one or more variables from the file.  Opening another data file then creates a new data view tab.  See the page on [Opening a Data File](Opening-a-Data-File.md) for more details.

### Question 2

CDAT shows the data value under the mouse cursor.

- [x] True

- [ ] False

**General feedback:** You can see the data value on the bottom right of the CDAT window when you pass your mouse cursor over valid pixels in the image.

### Question 3

The colors of the pixels in the data view are determined by the: (choose all answers that apply)

- [x] Enhancement range and function

- [x] Color palette

- [ ] Navigation tool bar

- [x] Data value at each location

**General feedback:** The pixel value is transformed to a color using the enhancement range, function, and palette. These are introduced on the [Color Palette and Enhancement Range](Color-Palette-and-Enhancement-Range.md) page.

### Question 4

The rendering order of data overlays cannot be altered.

- [ ] True

- [x] False

**General feedback:** The data overlays act like layers and can be rendered in any order.  See the [Data Overlays](Data Overlays.md) page for example screenshots and instructions for rearranging overlay rendering.

### Question 5

The <img src="../images/view_1_1_onscreen.png" alt="" height="24" align="top"> **Zoom to actual size** button:

- [x] Shows one pixel on the screen for every one pixel in the data

- [ ] Shows the actual location of the user on the data view

- [ ] Zooms out to fit the full data into the window

- [ ] Transfers the actual data value at the mouse cursor to the clipboard

**General feedback:** The <img src="../images/view_1_1_onscreen.png" alt="" height="24" align="top"> **Zoom to actual size** button is only mentioned briefly on the [Navigating Within the Data View](Navigating-Within-the-Data-View.md) page, but you can watch an example of it being used in the YouTube video linked from the bonus exercises on that page.

### Question 6

The data view can be exported to a PDF file.

- [x] True

- [ ] False

**General feedback:** The data view can be exported to various formats as presented on the [Data Export](Data-Export.md) page, including PDF.

### Question 7

The temperature of the Dead Sea in Israel is approximately:

- [x] 31.4°C

- [ ] 26°C

- [ ] 301.2 K

- [ ] 299.8 K

**General feedback:** You can find out the answer to this question by looking up the location of the Dead Sea on Google Maps, and then hovering your mouse cursor over the area in CDAT.  The Dead Sea has a very distinctive shape.  You may also need to convert from Kelvin to Celsius, depending on your CDAT settings.
