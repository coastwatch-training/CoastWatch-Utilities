### Question 3

Two scientific data formats used by CoastWatch are:

- [ ] SHP and DBF

  **Answer feedback:** SHP and DBF are common filename extensions relating to ESRI shapefiles.  Shapefiles contain point, line, and polygon coordinate data and DBF files contain their associated attributes.  Although these files can be used with the CoastWatch Utilities, they are not considered scientific data formats.

- [ ] PNG and JPEG

  **Answer feedback:** PNG and JPEG files are color image file formats, not scientific data formats.

- [ ] KML and XML

  **Answer feedback:** KML format files are used to annotate 2D and 3D geographic data.  XML format, while it can theoretically contain almost any type of data, is not considered a scientific data format.

- [x] HDF and NetCDF

**General feedback:** See the [CoastWatch Data Files](CoastWatch-Data-Files.md) page for information about what scientific data files contain.
